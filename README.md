En esta segunda versión del proyecto se ha refinado el menú interactivo y se ha completado la integración de todas las estructuras y clases solicitadas. 
Asimismo se implementaron los métodos de ordenamiento y búsqueda haciendo uso de las librerías nativas de Java (LocalDate, ChronoUnit) necesarias para la lógica de optimización de plazos.
El sistema se encuentra en una fase beta funcional y escalable, sentando una base sólida para futuras optimizaciones.
